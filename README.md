## 🪐 Overview

A premium, 3-page responsive web portfolio designed to wow clients while maintaining an aesthetic of stark minimalism. Inspired by the clean and technical "Nothing" brand aesthetics, it features a stark contrast design, subtle dot-matrix background touches, monospaced accents, and modern glassmorphism UI elements.

It's perfect for designers, video editors, and freelancers looking for an edgy but professional online presence.

## ✨ Key Features

- **🎨 Nothing-Brand Aesthetic:** Light theme, `#3bb371` green accents, subtle dot-matrix backgrounds.
- **🧊 Glassmorphism UI:** Premium translucent styling on navigation, forms, and pricing cards.
- **📱 Fully Responsive:** Adapts seamlessly to all screen sizes, from mobile to ultra-wide.
- **⚡ Fast & Lightweight:** Zero heavy frameworks; built with pure HTML, CSS, and Vanilla JS.
- **💸 Dynamic Pricing Grids:** Clean tabbed navigation for categorizing services (Logo, Poster, Video).
- **✉️ Pre-filled Contact Flow:** Clicking a specific pricing tier automatically selects that service in the contact form.

## 🛠️ Setup & Local Development

Want to run or edit this project locally? 

1. Clone the repository:
```bash
git clone https://github.com/devxign/Landing-Page-Temp2.git
```

2. Open the project folder:
```bash
cd Landing-Page-Temp2
```

3. Open `index.html` in your browser, or start a local dev server (e.g. using npx):
```bash
npx serve .
```
